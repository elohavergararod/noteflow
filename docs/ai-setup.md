# AI Setup

## Tool used

### Claude (claude.ai)

Claude is used as a **development assistant**, not as a code generator.
Its role in this project is limited to:

- Resolving specific errors and debugging
- Clarifying React Native concepts and documentation
- Suggesting solutions when stuck on a specific problem
- Reviewing code for obvious mistakes

## How it is used

Claude is consulted in specific moments during development — not as a
replacement for understanding the code. Every suggestion is reviewed,
understood and adapted before being added to the project.

The developer writes the code. Claude helps unblock specific problems.

## What Claude does NOT do in this project

- It does not define the architecture
- It does not decide which libraries to use
- It does not generate full components or screens
- It does not replace reading the official documentation

## Why this matters

Using AI as a crutch produces code the developer does not understand
and cannot maintain or debug independently. Using it as a consultant —
asking specific questions about specific problems — keeps the developer
in control of every decision.
